# Expense Tracker CLI

🚀 Final Project of My Internship Journey at SyntecxHub  
Domain: Python Programming  

## 📌 Description
A simple command-line Expense Tracker built using Python and SQLite.  
It allows users to:
- Add income and expense entries
- Store data in SQLite database
- Generate monthly summaries
- Visualize expenses using charts (matplotlib)

## 🛠️ Technologies Used
- Python
- SQLite
- Matplotlib

## ▶️ How to Run
1. Install Python
2. Install matplotlib:
   pip install matplotlib
3. Run:
   python main.py

## 📊 Features
- CLI based interface
- Monthly financial summary
- Expense visualization (bar chart)
